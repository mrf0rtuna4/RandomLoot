# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-25

### Added
- Initial public repository release of RandomLoot.
- Core random loot system for block breaks.
- Random mob spawn logic and special event effects.
- Configurable behavior via `config.yml`.
- Admin command set (`help`, `status`, `reload`, `toggle`).
- Initial project documentation and ARR license files.

### Context
- This plugin build was used during the Heypers 5th anniversary special event before open publication.
